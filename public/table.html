Table Tutorial
Table tags can be used to make a typical data sheet where you have rows and columns with information in each cell. In order to make this table we use the following code.
Example:
Name Birthday Age
Michael August 2nd, 1999 15
Marshall March 5th, 2015 1
"tr" tags are used to make a new row inside of the "table" tag. Inside those are "td", or table data, tags. These represent each cell in the row that the tag sits in. You can also use "th", or table header" tags. These will automatically center and bold the contents of cells with "th" tags instead of "td" tags.
